Lymphoma Classification System

This project introduces a web application for classifying Follicular Lymphoma FL and Mantle Cell Lymphoma MCL using a combination of Python, Flask, HTML, CSS, JavaScript, SQL with XAMPP, and the ResNet-18 deep learning model.

Bookingly

This web-based flight and hotel reservation system, built with HTML, CSS, PHP, and SQL, enables users to search, compare, and book flights and hotels seamlessly. It features user registration, secure payments, and an admin panel for managing listings and bookings. The system aims to provide a convenient and efficient platform for travel planning.
